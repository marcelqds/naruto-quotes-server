Naruto Quotes Api
====================
Simple app to generate random Naruto quotes using Node.js

sh```
    npm install naturo-quote-server-node
```

sh```
    npm run start
```
or set for include property in commandline `--port` 

example: 

sh```
    npm run start --port 8085
```
